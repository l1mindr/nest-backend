# Nest Backend

Enterprise-grade NestJS backend focused on scalability, maintainability and security.

## Features

- JWT Authentication
- Refresh Token Rotation
- Session Versioning
- RBAC
- PostgreSQL
- Redis
- Swagger
- Jest
- TypeORM
- Clean Architecture
